HTML

  • Each element needs a closing tag inside html code
  • The head element contains information about the webpage.
  • The body element represents the visible content shown to the user.
  • The meta element defines metadata
  • The head and meta elements are not visible to the user

CSS

  • CSS is a coding language used to style html code
  • Inline CSS is what developers use to style an element directly, by using a specific HTML tag
  • Can change the way HTML code is formatted, such as text and image size/color
  • A margin indicates how much space we want around the outside of an element.
  • A padding indicates how much space we want around the content inside an element.
  • Width defines how wide we want the element to be and height defines how tall we want it to be
  • When typing a class in CSS, a period must precede the class

Git

  • git status: checks what branch we are currently on
  • git checkout "branch name": switches the the branch you are working on to the branch stated
  • git checkout -b branch-name: creates a new branch and switches to it
  • The Main branch is the original branch where the main code lies
  • Feature branches are created to add and test code for new features, and are later merged with the main branch
  • Branches are updated and merged when a pull request is created and excepted

JavaScript

  • We can use a variable to store information
  • An array is a set of variables
  • 'const' is used for a fixed variable which cannot be changed later on, while 'let' allows for the variable to change
  • Control flow is important: keep in mind the order your code is in
  • If statements only work if the condition is true, whereas for loops work until it is told to stop. These can help with control flow
